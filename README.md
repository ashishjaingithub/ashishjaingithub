## Projects

### 🎙️ Veya — AI Voice Receptionist
An intelligent receptionist that handles inbound calls end-to-end: qualifies leads, books appointments, updates CRM, routes callers, and alerts the team. 

**Stack:** Node.js · Fastify · PostgreSQL · BullMQ · Retell AI · HubSpot

---

### 🎯 Hunter — AI Sales Rep
A fully-agentic sales rep that runs daily — sourcing leads, writing personalized outreach, processing replies, and reflecting on what's working. Humans stay in the loop for anything that matters.

**Stack:** TypeScript · Next.js · LangGraph · PostgreSQL

---

### 📍 GramRover — Instagram Location Intelligence
Analyzes Instagram exported data to surface behavioral and geographic patterns. Useful for exploring your own instagram activity and looking at interesting insights. 

**Stack:** Next.js · Dexie · Vitest

---

### 💼 Appy — Job Application Pipeline
Define your profile once. Appy fetches matching jobs on a schedule, scores them against your qualifications, pre-fills application with answers from your profile, and queues them for your review.

**Stack:** Next.js · Ollama · SQLite · Chrome Extension

---

### 📸 AI Dose — Instagram Content Pipeline
Generates one ready-to-post Instagram image and caption per day. Runs fully automated via a scheduled pipeline.

**Stack:** Python · Anthropic Claude · Pillow

---

### 🛋️ Crib Sheet — Room Personality Analyzer
Upload a photo of your room. Get a detective-style personality profile back. Built on Moondream 2 vision model and Claude Haiku.

**Stack:** Next.js · Python FastAPI · Moondream 2

---

### 🏕️ SummerCampSync — AI Camp Finder
Finds summer camps that match a child's interests, schedule, and location using AI-powered search and filtering.

**Stack:** Next.js · Gemini

---

### 🧩 Cracked — Daily AI Puzzle Game
A daily puzzle game where the answer changes every day using a deterministic seed. Think Wordle meets Escape rooms. 

**Stack:** Next.js · Vitest

## Open Source

### 🩻 [clawdoc](https://github.com/ashishjaingithub/clawdoc)
Diagnoses AI agent sessions from JSONL logs. Detects 14 behavioral patterns — retry loops, context exhaustion, cost spikes, tool misuse, and more — and prescribes fixes. An [OpenClaw](https://github.com/openclaw) skill that works with any agent producing JSONL logs.

**Stack:** Bash · jq
